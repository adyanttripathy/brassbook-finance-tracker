# 💰 BrassBook — Personal Finance Tracker

> *Every penny accounted · every win rewarded*

BrassBook is a comprehensive personal finance tracker built with React. It helps individuals manage day-to-day expenses, savings goals, budgets, and rewards you with **brass points + shareable badges** whenever you save more than the previous month.

---

## ✨ Features

### 📊 Overview Dashboard
- Net balance, monthly income, spending, and savings at a glance
- 6-month income/expense/savings trend chart
- Category spending pie chart
- "Beat last month" progress meter with live points projection

### 💳 Money Ledger
- Log **income** and **expenses** across 14+ categories
- Browse by month, filter by type, search by note
- Edit or delete any entry

### 🎯 Budget Tracker
- Set monthly spending limits per category
- Live progress bars — amber at 85%, red when over

### 🏦 Savings Goals
- Named goals with emoji, target amount, and optional deadline
- Track progress visually, fund goals manually

### 🧠 Money Coach
- 10+ dynamic tips from your own ledger
- 50/30/20 rule, savings rate, latte factor, subscription audit, MoM trends

### 🏅 Rewards & Badges
- Earn 100+ pts every month you save more than the month before
- Mint 8 guild medallions (Copper Sprout → Mythic Money Master)
- Share badges on Twitter/X, WhatsApp, Telegram, or download PNG

---

## 🚀 Getting Started

```bash
git clone https://github.com/adyanttripathy/brassbook-finance-tracker.git
cd brassbook-finance-tracker
npm install && npm run dev
```

**Dependencies:** React 18, Recharts
